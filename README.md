# Tetrix
unicode console tetris

## Requirements
`.NET Core` v1.1 or higher (downloadable form [here] (https://www.microsoft.com/net/download/core))

## Installation
```bash
# Download using git
git clone https://github.com/stoyandimov/tetrix

# Restore dependencies 
cd tetrix/src
dotnet restore

# Build
dotnet build
```

## Run
```bash
# cd into tetrix/src
dotnet run
```
